# Porsche 911 Premium Three.js Concept

A static, cinematic landing-page concept showcasing the Porsche 911 with a premium, luxury-first visual language.

## Stack
- HTML
- CSS
- JavaScript (ES modules)
- Three.js

## Run locally
Open `index.html` with a local static server for module support, for example:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Optional real car model
The scene tries to load `assets/models/porsche_911.glb`. If missing, it uses a stylized fallback car mesh so the experience still works.
